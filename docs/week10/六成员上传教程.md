# SG CampusSwap — 六成员上传教程

> 上传完成后项目达到可部署状态，手机浏览器即可访问。
> 全程约 15 分钟。严格按顺序执行，前一人完成后下一人再开始。

---

## 准备：确认 GitHub 仓库

六个人都需要：
1. 打开 https://github.com/LuoX11a/SG_CampusSwap
2. 确认当前在 `develop` 分支
3. `git pull origin develop` 拉取最新代码

---

## 第 1 步：Wang Bowei（后端负责人）⏱ 约 3 分钟

> 📦 上传：API 路由 + Docker + 配置文件
> ⚠️ 必须第一个上传，后面所有工作依赖这些文件

```
你的文件夹位置：week10/Wang-Bowei/
```

### 1.1 复制 API 路由文件
```
将 Wang-Bowei/backend/app/api/v1/ 下面的 8 个文件
复制到 → 项目根目录/backend/app/api/v1/
（如果 v1 文件夹不存在，先创建它）
```

8 个文件清单：
- `__init__.py`
- `auth.py`（注册/登录/验证）
- `items.py`（物品 CRUD）
- `users.py`（用户资料）
- `reviews.py`（评价系统）
- `search.py`（搜索筛选）
- `upload.py`（图片上传）
- `chat.py`（实时聊天）

### 1.2 覆盖 main.py
```
将 Wang-Bowei/backend/app/main.py
复制到 → 项目根目录/backend/app/main.py（覆盖原文件）
```

### 1.3 复制 Docker 文件
```
Wang-Bowei/Dockerfile        → backend/Dockerfile
Wang-Bowei/docker-compose.yml → 项目根目录/docker-compose.yml
Wang-Bowei/.env.example       → backend/.env.example（覆盖）
```

### 1.4 提交到 GitHub
```bash
git add backend/app/api/v1/ backend/app/main.py backend/Dockerfile docker-compose.yml backend/.env.example
git commit -m "feat: add API routes, Docker config, and health check"
git push origin develop
```

## 第 3 步：Renxian Tang（前端负责人）⏱ 约 3 分钟

> 📦 上传：Next.js 完整项目（含移动端适配）
> ⚠️ 必须在后端两人之后上传

```
你的文件夹位置：week10/Renxian-Tang/
```

### 3.1 复制整个 web 文件夹
```
将 Renxian-Tang/web/ 下面的所有内容
复制到 → 项目根目录/project/web/
（如果 project/web 不存在，先创建文件夹）
```

包含内容：
| 类别 | 内容 |
|------|------|
| 配置文件 | `package.json`, `next.config.js`, `tailwind.config.ts`, `tsconfig.json`, `postcss.config.js`, `.env.local` |
| 核心页面 | `layout.tsx`（响应式布局）, `page.tsx`（首页）, `globals.css` |
| 认证页面 | `login`, `register`, `verify-email` |
| 功能页面 | `browse`, `item/[id]`, `sell` |
| 组件 | `CategoryChips`, `ItemCard`, `SearchBar`, `Sidebar`, `SkeletonCard`, **`MobileBottomNav`**（底部导航） |
| 状态管理 | `auth-store`, `chat-store`, `filter-store`, `item-store` |
| 工具库 | `api-client`, `format`, `types` |
| **PWA** | `manifest.json`, `sw.js`, `icons/` |

### 3.2 提交到 GitHub
```bash
git add project/web/
git commit -m "feat: add Next.js frontend with mobile responsive + PWA"
git push origin develop
```

## 上传完成！验证

```

├── backend/
│   ├── app/
│   │   ├── main.py              ← Wang Bowei 更新

│   │   └── services/            ← Wang Xu (3 个 service)
│   ├── tests/                   ← Wang Xu (10 个测试)
│   ├── Dockerfile               ← Wang Bowei
│   └── .env.example             ← Wang Bowei 更新
├── docker-compose.yml           ← Wang Bowei
├── project/
│   └── web/                     ← Renxian Tang + Jiahai Xiong (完整)
├── docs/
│   ├── week8/                   ← Junliang Li + Huang Hongrui
│   ├── week9/                   ← Junliang Li + Huang Hongrui
│   ├── week10/                  ← Huang Hongrui
│   └── DEPLOY_GUIDE.md          ← Huang Hongrui
└── week10/                      ← 本文件夹（不上传）
```

运行验证：
```bash
# 后端
cd backend
pip install -r requirements.txt
uvicorn app.main:app --reload
# 浏览器打开 http://localhost:8000/docs

# 前端
cd project/web
npm install
npm run dev
# 手机连同一 WiFi，浏览器打开 http://你的IP:3000
```
