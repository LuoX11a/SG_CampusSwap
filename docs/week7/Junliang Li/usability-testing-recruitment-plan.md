## 4. Moderator Script

```
═══════════════════════════════════════════
         MODERATOR SCRIPT
═══════════════════════════════════════════

[SETUP — Before participant arrives]
□ Test device charged and ready
□ Expo Go app installed, app loaded
□ Screen recording app ready
□ Consent form printed
□ Task sheet printed
□ Observation notes template open
□ Voucher ready

[INTRODUCTION — 2 min]
"Hi [Name], thanks for coming! I'm [moderator name].

We're testing a new campus marketplace app called SG
CampusSwap. We want to see how real students use it so
we can make it better.

Important things to know:
1. We're testing the APP, not you. There are no wrong
   answers. If something is confusing, that's the
   app's fault, not yours.
2. Please THINK ALOUD as you go. Say what you're
   looking at, what you expect to happen, and if
   anything surprises you.
3. I can't help you during the tasks (even if you get
   stuck) — that's how we learn what needs fixing.
4. You can stop at any time.

Any questions before we start?

[Have them sign the consent form]

[Start screen recording]

[TASKS — 15–18 min]
"Here's the first task..."

[Hand them the task sheet, one task at a time]

[Observe silently. Take notes on:
 - Where they hesitate or get confused
 - What they say aloud
 - Any errors they encounter
 - Whether they complete the task
 - Time taken per task]

[POST-TASK — 5 min]
"Great, that's all the tasks. A few quick questions..."

[Administer SUS + custom questionnaire]

"What was the most confusing part?"
"What would make this app better for you?"

[WRAP-UP]
"Thank you so much! Here's your voucher.

Your feedback will directly help us improve the app
for thousands of students. We really appreciate it."

[Stop recording]
[Save recording file: US-[Participant#]-[Date].mp4]
```
